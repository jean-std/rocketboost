RocketBoost

Aqui você irá acompanhar a contrução de nossa base de lançamento e nosso foguete, todos eles foram utilizados com materiais de fácil acesso e seguindo nosso protótipo feito atravéz do SketchUp


Construção da Base

Etapas do desenvolvimento

  1. Para começar separamos os materiais necessários para a construção,oque são eles:
    - 2 canos pequenos de PVC
    - 1 T de PVC
    - 2 Joelhos de PVC
    - 2 Tampas de PVC
    - 1 Cola Adesiva Plástica

  2. Após separar os materiais, vamos tirar algumas medidas para fazer os cortes necessários paraontar a base


  3. Com os cortes feitos,o próximo passo é lixar para remover as imperfeições e colocar os materiais seguindo nosso protótipo

  4. Com todos materiais cortados, medidos e colados, obtivemos nossa base de lançamento, deixando o mais próximo de nosso projeto inicialmente modelado

Desenvolvimento do Foguete

Com nossa base de lançamento finalizada partimos para a criação do foguete


Etapas do desenvolvimento

  1. Inicialmente vamos separar os materiais necessários, são eles:
    - 1 Garrafa PET inteira
    - 1 Parte superior de uma garrafa PET
    - 1 Base de uma garrafa PET
    - 3 Partes de plástico para nossa base

  2. Com os materiais separados, vamos iniciar a montagem do nosso foguete, toda a construção dele você pode acompanhar no vídeo abaixo

  3. Pronto, agora temos em mãos nosso foguete, construído co. materiais de fácil acesso e de uma forma simples
